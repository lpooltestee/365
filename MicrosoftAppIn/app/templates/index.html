Como Instalar o Add-in
  1. Acesse o Admin Center do Microsoft 365.
  2. Navegue até Configurações > Aplicativos integrados > Add-ins.
  3. Clique em Implantar add-in e selecione Carregar add-in personalizado.
  4. Faça upload do arquivo manifest.xml.
  5. Selecione os usuários para os quais o add-in deve ser disponibilizado.
  6. Clique em Implantar.
Como Usar o Add-in

Ao criar um novo e-mail no Outlook, o add-in irá inserir automaticamente a assinatura corporativa.

Se necessário, você também pode clicar no botão Inserir Assinatura na faixa de opções.

Recursos
  • Sincronização com Microsoft 365 - Importe automaticamente os dados dos usuários.
  • Editor de Templates - Crie e personalize facilmente os templates de assinatura.
  • Área Administrativa - Gerencie usuários e assinaturas de forma centralizada.
  • Inserção Automática - A assinatura é automaticamente inserida ao criar um novo e-mail.
